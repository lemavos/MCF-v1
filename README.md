## Como rodar

1. Instale o Python 3.x

2. Clone o repositorio e entre na pasta do projeto
   ```
   git clone https://github.com/lemavos/MCF-v1.git
   cd MCF-v1
   ```

3. Abaixe as dependecias
   ```
   pip install -r requirements.txt
   ```

4. Rode o projeto
   ```
   python mcf_hub.py
   ```
